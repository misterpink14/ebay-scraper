# es6


### Up and Running 

`git clone https://github.com/ProvoJS/es6.git`

`npm i`

### Run Client

`npm run client`

go to `http://localhost:8080/` in browser

### Run Server

`npm run server`

go to `http://localhost:3000/` in browser
